This is the backend + frontend project for a service booking application where customers book services and technicians can view their matched bookings and receive email notifications.

 <!-- features -->

Customer registration & login  
Technician registration & login  
 Booking services with date & slot  
 Technician dashboard showing only matched bookings  
 Email notifications to technicians on new bookings  
 Backend REST APIs  
 Frontend UI in React

---

<!-- tech stack -->

| Layer    | Technology        |
| -------- | ----------------- |
| Backend  | Node.js, Express  |
| Database | MongoDB, Mongoose |
| Frontend | React, TypeScript |
| Email    | Nodemailer        |
| Styling  | Tailwind CSS      |

Node.js (v16+)  
 NPM  
 MongoDB (Local )  
 Google account with App Password (for email)
